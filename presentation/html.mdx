### Arrow Functions

##### Arrow functions allows a short syntax for writing function expressions.

##### You don't need the function keyword, the return keyword, and the curly brackets.

```Javascript
// ES5
var x = function(x, y) {
  return x * y;
}

// ES6
const x = (x, y) => x * y;
```

### Spread Operator
---
```javascript
var array = ["a1","a2","a3"]
console.log(array) //["a1", "a2", "a3"]
console.log(...array) //a1 a2 a3
````

---

```javascript
nums1 = [1, 2, 3]; // [1, 2, 3]
nums2 = [4, 5, 6]; // [4, 5, 6]
console.log(nums2); // [4, 5, 6]
console.log(nums1, nums2); // [1, 2, 3] (3) [4, 5, 6]
console.log(...nums1, ...nums2); //1 1 2 3 4 5 6
```

---

```javascript
var nums = [1, 2, 3];

function addNums(a, b, c) {
  return a + b + c;
}

addNums(...nums); //6
```

## Asynchronous JavaScript

---

### Asynchronous Code Can Start now finish later

#### Like Fetch Data from Youtube, twitter,...Api's

---

#### synchronous code

##### Javascript code runs on a single thread(cand do 1 thing at a time)

```javascript
var article = readSync(article);
console.log(article);
var auther = readSync(auther);
console.log(auther);
```

---

#### Asynchronous code

##### runs code on a multiple thread(cand do 1 thing at a time)

```javascript
readAsync(article, function() {
  console.log(article);
});

readAsync(autor, function() {
  console.log(auther);
});
```
